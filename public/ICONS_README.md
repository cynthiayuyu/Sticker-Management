# PWA Icons

此專案需要以下圖示檔案來完整支援 PWA 功能：

## 需要的檔案：

1. `icon-192.png` - 192x192 像素的 PNG 圖示
2. `icon-512.png` - 512x512 像素的 PNG 圖示

## 建議製作方式：

### 線上工具：
- https://realfavicongenerator.net/ （自動生成各種尺寸）
- https://www.favicon-generator.org/

### 設計建議：
- 使用專案主色調：紫色 #7D7489
- 背景色：米白 #F9F8F6
- 可以使用優雅的字母 "L" 或 "A" （L'Atelier）
- 風格：極簡、優雅、現代

### 暫時解決方案：
目前可以使用任何 192x192 和 512x512 的圖片作為佔位符。
圖示不影響功能，只影響安裝後的外觀。

## 快速製作圖示：

如果您有 logo 或想要的圖案，可以：
1. 用 Canva / Figma 製作 512x512 的圖片
2. 匯出為 PNG
3. 用線上工具縮小到 192x192
4. 將檔案命名為 `icon-192.png` 和 `icon-512.png`
5. 放在 `public/` 資料夾
